# Ringle앱 > Webinar 탭 클론


## 1. 리포지토리 클론
```bash
git clone https://github.com/kwonh11/clone-rg.git
```
---

## 2. 종속성 모듈 설치
```bash
yarn install
```
or
```bash
npm install
```

---


## 3-1. IOS 실행

```bash
cd ios && pod install && cd .. && yarn run ios
```
or

```bash
cd ios && pod install && cd .. && npm run ios
```

## 3-2. Android 실행
```bash
cd ios && pod install && cd .. && yarn run android
```
or
```bash
cd ios && pod install && cd .. && npm run ios
```
